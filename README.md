# Cybersecurity Awareness Chatbot

A console-based chatbot that educates users about cybersecurity best practices including password safety, phishing detection, and safe browsing.

## Features
- Voice greeting on startup
- ASCII art logo display
- Personalized responses using user's name
- Input validation (handles empty responses)
- Typing effect for realistic conversation
- Colored console interface
- Cybersecurity topic responses

## How to Run
1. Clone this repository
2. Open in Visual Studio
3. Ensure `greeting.wav` is in the output directory
4. Press F5 to run

## Topics You Can Ask About
- Password safety
- Phishing scams
- Safe browsing habits
- How the bot works
